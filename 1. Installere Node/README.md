# Installere Node

https://nodejs.org/en/download

De fleste prosjekt i Nav er Nodejs basert. Det finnes løsninger som bruker [Deno](https://deno.com/) og [Bun](https://bun.com/), men gode gamle Node har de siste årene fått flere flere forbedringer og er fortsatt go-to javascript runtime for de fleste.

## Version manager

Jobber du i flere prosjekter, hender det at du må endre Nodejs version mellom dem. Da anbefales det en "version manager" for node. Node anbefaler ["nvm"](https://github.com/nvm-sh/nvm) eller ["fnm"](https://github.com/Schniz/fnm) for dette.
